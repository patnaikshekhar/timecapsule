Summary

summarising the underlying technologies used in the project and highlighting key achievements

HTML

Bootstrap elements have been used such as using glyphicons. Text and adding buttons over text was a bit challenging but was accomplished in Main_Home.html. Making the image responsive in Main_Home.html took some time but was successfuly achieved. Used floats and clear in Home_sent_OLD.html to align. Questinnaire.html uses a lot of different inputs

CSS

Used CSS in external css sheets and also within HTML. This has been used for alignment, padding, margin, font size, font, color etc. Shadows to boxes in Login.html and Sign_Up has been applied to enhance and give a 3d effect. Used multiple stylesheets in Main_Home.html

Javascript

Used Javascript in Create1.html. When user clicks upload, two buttons appear- Save and Upload. In HTML the visibility of the button is hidden and once the user clicks upload the two buttons appear. In Create.html an alert box pops up when you click close

Responsiveness

Both from scratch and bootstrap is used in the website. Media queries are used throughout to make the webpages responsive. Bootstrap is used in pages like Main_Home.html where content in rows is divided into blocks using class col-*-*.
